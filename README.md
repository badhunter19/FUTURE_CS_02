# FUTURE_CS_02 - Phishing Email Detection & Awareness System

**Cyber Security Task 2 (2026)**  
**Submitted to:** Future Interns  
**Role Simulated:** Security Analyst (SOC / GRC / Security Awareness)

---

## Project Overview

This project demonstrates real-world Security Analyst skills by analyzing five genuine phishing email samples, performing detailed header analysis, identifying phishing indicators, classifying risk levels, and producing a professional **employee awareness report**.

Phishing continues to be one of the most prevalent and damaging cyber threats faced by organizations. This report emphasizes the human element of security — helping employees recognize phishing attempts through clear examples, simple explanations, and practical prevention strategies.

## Objective

- Analyze real phishing emails (including the official task sample)
- Perform technical email header analysis (SPF, DKIM, DMARC)
- Identify behavioral and technical red flags
- Classify each email’s risk level
- Explain attacks in simple, non-technical language
- Create a clean, ready-to-use awareness document for businesses
